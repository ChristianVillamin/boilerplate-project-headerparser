Request Header Parser Microservice

A freeCodeCamp API Project by @villamin_c. Link to GitHub.

Usage

GET /api/whoami
Responds with JSON format of your IP Address, Language Preference, and System Infos

Example Output

{"ipaddress":"159.20.14.100","language":"en-US,en;q=0.5", "software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}

User Story

I can get the IP address, preferred languages (from header Accept-Language) and system infos (from header User-Agent) for my device.
